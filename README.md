## Introdução

A educação é elemento-chave no desenvolvimento social e econômico. No Brasil, diferentes estados enfrentam realidades distintas, o que torna essencial estudar os fluxos escolares como: promoção, repetência e evasão. As taxas de transição disponibilizadas pelo INEP¹ capturam a proporção de alunos que efetivamente seguem para a série subsequente, sendo instrumentos úteis para inferir movimentos de evasão ou interrupção do percurso escolar.
Este projeto propõe analisar a trajetória dessas taxas de transição por estado ao longo dos anos, com especial atenção ao ensino médio, com o objetivo de identificar padrões, rupturas e disparidades entre os estados.
